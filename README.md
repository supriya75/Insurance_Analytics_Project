# Insurance Analytics Project

## Project Overview
This project demonstrates end-to-end insurance claims analytics:
- Data cleaning and validation
- KPI calculations: Loss Ratio, SLA breach, High-risk claims
- Aggregation and regional summaries
- Anomaly detection
- Visualizations for insights

## Tech Stack
- Python: pandas, numpy, matplotlib
- SQL: MySQL queries for insurance analysis
- Power BI: dashboard visualizations

## Dataset
- Synthetic insurance claims dataset (`data/claims_data.csv`)
- Includes: claim_id, policy_id, region, claim_type, claim_amount, premium, claim_status, processing_days

## Key Insights
- High-risk claims and SLA breaches identified
- Regional claim summaries with Loss Ratios
- Running totals and claim ranking using SQL
- Visualizations: claim distribution, SLA breach counts, loss ratio by claim type

## How to Run
1. Open `claims_analysis.ipynb` in Jupyter Notebook
2. Load `claims_data.csv` in the `data/` folder
3. Run all cells sequentially to view cleaned data, KPIs, and visualizations
